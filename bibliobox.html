¿Qué es una Bibliobox?

Bibiliobox es un proyecto de cultura libre, es una biblioteca/isla/virtual/portable de libros

Es un dispositivo al que podés conectarte por wifi y explorar una biblioteca colectiva localizada, desconectada de internet

Piratebox

computadora nodo para intercambiar información

Calibre

Software libre que permite administrar una biblioteca virtual de documentos / convertirlos a diferentes formatos

Biblioteca-guerrilla

El script que permite crear una web estática desde una biblioteca Calibre

Bibliobox

Es una instancia portable donde se incluye el sitio creado por bibliotega-guerrilla y es servido por un servidor web configurado en la raspberry pi, como dispositivo de punto de acceso
